# **App Name**: MapCorrect

## Core Features:

- Location Pinning: Interactive map with location pinning for error reports.
- Error Type Selection: Predefined error type selection (wrong names, missing roads, etc.).
- Report Details: Detailed description and photo upload for each report.
- Report Submission: Submission of error reports for community review.
- Audit Trail Tab: Tab sections in settings for easy audit trails and action tracking by type
- Impact Analyzer: AI tool that will analyze the reports to categorize by impact.

## Style Guidelines:

- Primary color: A fresh green (#7CFC00) to inspire confidence and trustworthiness in the app’s error reporting system.
- Background color: Off-white (#FAFAFA), subtly desaturated from the green, provides a clean, neutral backdrop that doesn't distract from the map and other content.
- Accent color: A light green (#98FF98), analogous to the primary color, highlights interactive elements.
- Font: 'Inter', a grotesque-style sans-serif font. Good for headers or body text.
- Simple, clear icons to represent error types.
- Mobile-first, responsive design.
